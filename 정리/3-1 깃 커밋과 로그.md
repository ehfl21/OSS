# 깃 3 영역
## 작업 영역
### working directory(folder), working tree
* 탐색기 상의 폴더 하부
## 스테이징 영역
### staging area, index
* 저장소의 .git 폴더의 파일 index
## 깃 저장소(git repository)
### 저장소
* 저장소의 .git 폴더의 여러 정보

# 파일 생성 후 add, commit 수행
### $ git add
* 파일을 Working tree => Staging Area로 이동(복사)
### $ git commit
* Staging Area에 준비된 파일 => Git repository(버전관리)로 이동(복사)
